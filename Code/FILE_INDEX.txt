================================================================================
GENERATED OUTPUTS (After running)
================================================================================

The following files will be generated after running the code:

Models:
-------
- best_baseline_model.pth (~1 MB)
- best_causal_model.pth (~1.2 MB)

Visualizations:
--------------
- causal_decomposition.png
  └─ 6-panel visualization showing RUL breakdown for sample predictions
  
- counterfactual_analysis.png
  └─ 4-panel visualization:
     • Condition change effects heatmap
     • RUL change distribution
     • Wear reduction impact scatter
     • Factual vs counterfactual comparison

- training_curves.png
  └─ 2-panel showing training/validation losses for both models

- predictions_comparison.png
  └─ 3-panel comparison:
     • Baseline predictions scatter
     • Causal predictions scatter
     • Error distribution histogram

Data Files:
----------
- counterfactual_results.csv
  └─ Detailed counterfactual scenario results with columns:
     sample_id, original_condition, new_condition, factual_rul,
     counterfactual_rul, delta_rul, intervention_type

Reports:
-------
- summary_report.txt
  └─ Comprehensive results summary including:
     • Data statistics
     • Model performance metrics
     • Causal insights
     • Counterfactual findings
     • Key contributions
     • Generated outputs list
